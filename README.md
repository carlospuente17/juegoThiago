# Mundo Thiago

Un juego educativo móvil interactivo diseñado especialmente para niños de 6 años en Ecuador. Completamente construido en HTML, CSS y JavaScript puro.

## Características
- **5 Materias Educativas:** Lenguaje, Matemática, Ciencias Naturales, Estudios Sociales y Arte.
- **Diseño Mobile-First:** Experiencia tipo app nativa iOS, con animaciones fluidas, safe-areas, y navegación inferior.
- **Micro-interacciones:** Retroalimentación visual inmediata con animaciones CSS (confeti, estrellas, medallas).
- **Modos de Juego:** Juega por rondas o contra el reloj.
- **Sin Dependencias:** Pesa apenas unos kilobytes. Sólo usa fuentes e íconos de Google Fonts.

## Despliegue en GitHub Pages

Dado que todo el proyecto se compone de un solo archivo `index.html`, publicarlo en GitHub Pages es extremadamente sencillo:

1. Crea un nuevo repositorio en tu cuenta de GitHub (ej. `mundo-thiago`).
2. Sube el archivo `index.html` a la raíz del repositorio.
3. En GitHub, ve a **Settings** (Configuración) de tu repositorio.
4. En el menú lateral izquierdo, haz clic en **Pages**.
5. Bajo **Source / Build and deployment**, selecciona `Deploy from a branch`.
6. En **Branch**, selecciona `main` (o `master`) y la carpeta `/ (root)`, luego haz clic en **Save**.
7. ¡Listo! En un par de minutos, tu juego estará en vivo en tu enlace de GitHub Pages (ej. `https://tu-usuario.github.io/mundo-thiago`).

## Tecnologías Utilizadas
- HTML5 (Estructura)
- CSS3 (Estilos, Flexbox, Keyframes Animations)
- JavaScript Vanilla (Lógica del DOM y estado de juego)
- Fuentes: Nunito & Material Symbols Rounded (Google Fonts CDN) No se utilizaron emojis según requerimientos.
